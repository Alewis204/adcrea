# Adcrea Platform Specifications

## Export Dimensions & Formats

### Facebook / Instagram
- **Feed (square)**: 1080x1080px, JPG/PNG/MP4
- **Feed (portrait)**: 1080x1350px, JPG/PNG/MP4
- **Story / Reels**: 1080x1920px, MP4
- **Carousel**: 1080x1080px per card
- **Collection**: 1200x628px

### Google
- **Shopping / Display**: 1200x628px, JPG/PNG
- **Performance Max**: 1200x1200px, 1200x628px, 600x600px
- **Discovery**: 1200x628px, 600x600px

### Amazon
- **Main image**: 1000x1000px (or 500x500px min), pure white background, JPG/PNG
- **Infographic**: 1000x1000px, JPG/PNG
- **Video**: 1920x1080px, MP4

### Shopify
- **Product**: 2048x2048px, JPG/PNG/WebP
- **Collection**: 1024x1024px
- **Hero / Slideshow**: 1920x800px

### TikTok
- **Feed**: 1080x1920px, MP4
- **Story**: 1080x1920px, MP4/PNG
- **Carousel**: 1080x1920px per slide

### LinkedIn
- **Feed (square)**: 1080x1080px
- **Feed (portrait)**: 1080x1350px
- **Carousel**: 1080x1080px per slide
- **Document ad**: 1920x1080px

## Commercial License
All outputs include a standard commercial license grant. Mark outputs with metadata indicating license type.

## File Naming Convention
`{product_name}_{format}_{platform}_{timestamp}.{ext}`
e.g. `blue-sneakers_static-ad_facebook_1692000000.png`

## Watermark (Free Tier)
Low-res exports (50% of full res) with semi-transparent "Adcrea" watermark centered.
